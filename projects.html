Selected engineering work — production LLM systems, multi-agent platforms, and infrastructure. More on GitHub.

  • Multi-Agent LLM Evaluation Platform

    Python · GPT-4o · Asyncio
    Summoner, New York — AI Engineer

    An LLM-driven platform that ingests user context, generates domain-agnostic challenge questions with logical reasoning pathways, synthesizes AI personas, and orchestrates real-time timed challenge sessions. Answers are evaluated with GPT-4o structured logic extraction and embedding cosine-similarity scoring for automated partial credit. Includes a configurable multi-factor scoring system, cost-observability safeguards with real-time USD tracking and budget ceilings, semaphore-bounded async concurrency, and a state-machine protocol letting persona agents request peer hints through async message queues.

  • Server-Side Pagination Re-architecture

    SQL · Redis · Microservices
    Antares Capital, New York — Software Engineer

    Re-architected a legacy service from client-side full data loading to server-side pagination with database-driven SQL queries — reducing transfer size from ~50MB to ~200KB per request and page loads from 10–30s to ~1–2s. Introduced pagination-friendly indexing and Redis-based distributed caching, enabling stateless services and horizontal scaling across API servers.
